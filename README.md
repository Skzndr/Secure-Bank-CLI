Secure Bank CLI

Description

Secure Bank CLI est une application Python en ligne de commande simulant un système bancaire sécurisé.

Le projet démontre :

- Authentification sécurisée via hash SHA-256
- Protection contre brute force (3 tentatives maximum)
- Gestion de dépôts et retraits
- Logging structuré avec niveaux (DEBUG, INFO, WARNING, ERROR)
- Séparation modulaire du code



Technologies utilisées

- Python 3
- hashlib
- logging



Concepts démontrés

- Hashing de mot de passe
- Gestion des erreurs
- Séparation des responsabilités (authentification / opérations)
- Architecture modulaire
- Gestion des environnements (DEV / PROD)
